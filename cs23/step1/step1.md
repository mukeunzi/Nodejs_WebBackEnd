# STEP1. 디지털 논리회로

----
## 배경지식
컴퓨터 하드웨어를 구성하는 디지털 논리회로. 디지털 시스템을 구성하는 기본이 되는 3가지 부품을 기본 논리 게이트라고 한다.

릴레이Relay 부품(입력, V[전압], 출력)? 입력에 전기가 흐르면 전자석 스위치 동작 - V 전압 - 출력으로 전달. 다시 말해서 입력 전압이 있는 상태를 1 - true - 참이라고 한다면, 출력에도 전압이 흐르고 1 - true - 참 상태가 된다.

----
## 1) AND 게이트

입력을 두 개 받고, 두 개 입력이 모두 true 일 때만 전구에 불이 들어오는 true 상태가 된다.

----
## 1) OR 게이트

입력을 두 개 받고, 두 개 입력중에 하나라도 true 상태이면 전구에 불이 들어오는 true 상태가 된다.

----
## 1) INVERTOR(or NOT)

입력 상태와 반대로 뒤집힌 결과가 나온다. true 는 false, false 는 true.

----
## 이진수(Binary)

전원이 꺼진 상태와 켜진 상태를 각각 false 와 참 true 로 표현한다고 가정한다. 컴퓨터 공학에서는 논리대수(Boolean Algebra/방식)를 창안한 영국 수학자 조지 부울 이름을 따서 부울린(Boolean) 타입이라고 부른다. 부울린 타입은 논리대수 기본 타입이면서 디지털 시스템을 구성하는 하드웨어와 소프트웨어의 가장 기본적인 타입이다.

모든 전자부품은 전기가 흐르냐 흐르지 않느냐로 부울린 타입을 표현할 수 있고, 이 개념을 확장해서 2진수(Binary)로 모든 숫자(10진수..)를 표현할 수 있다.

1비트(bit)는 0과 1을 표현할 수 있으며 2개 (2의 1승) 값을 표현할 수 있다.(2의 1승)
따라서 8비트는 0~255까지 표현할 수 있으며 총 256개의 값을 표한할 수 있다.(2의 8승)
8비트를 1바이트(Byte)라고 부른다.
